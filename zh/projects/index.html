使用深度学习进行面部表情识别#

2024年10月 至 2024年12月，Python / Machine Learning / Deep Learning

GitHub: https://github.com/zhouyibin8/facialexpression

  • 使用了FER2013（Facial Expression Recognition 2013 Dataset）数据集。
  • 使用Python，构建并评估了多种机器学习模型，包括逻辑回归、支持向量机（SVM）和随机森林。
  • 开发了基于卷积神经网络（CNN）的深度学习模型，并实现了ResNet和Inception等高级架构。
  • 进行了超参数调优，并使用数据增强、dropout和批量归一化等技术来提高模型性能。
  • 通过使用预训练模型进行迁移学习，显著提高了准确率。
  • 结果表明，优化后的深度学习模型在表情识别准确率上显著优于传统的机器学习模型，最高准确率达到了92.65%。

学生辍学率预测模型#

2024年9月 至 2024年10月，Python / R / Machine Learning

GitHub: https://github.com/zhouyibin8/drop

  • 利用R和Python进行数据清洗和分析。
  • 应用了数据清洗技术，如处理缺失值和创建新变量用于分析。
  • 构建并评估了多种机器学习模型，包括逻辑回归、决策树、随机森林、梯度提升和XGBoost。
  • 进行了超参数调优，并采用交叉验证技术优化模型性能。
  • 使用多项式特征增强的XGBoost模型达到了96%的准确率。

基于爬虫的网页数据挖掘#

2023年1月 至 2023年6月，Python

GitHub: https://github.com/zhouyibin8/wp

  • 该项目利用Python的爬虫技术收集网页信息，以帮助公司进行战略决策。主要目标是收集澳门地区大众点评上的竞争对标餐厅的相关信息，为市场分析和决策提供数据支持。
  • 收集的数据包括餐厅的基础信息、评分、用户评论以及相关菜品等，通过分析这些信息，可以更好地了解市场动态和竞争情况。
  • 在数据收集完成后，使用百度翻译将上述内容自动翻译成英文，帮助外国同事理解和使用这些数据，促进了团队的国际合作与沟通。

大师班: https://www.must.edu.mo/fhtm/activities

  • 该活动邀请了三位泰国米其林主厨来到澳门，与永利皇宫的主厨共同打造了一场为期三个月的泰式美食节。活动内容包括高端晚宴、自助餐和大师班，旨在为宾客提供地道的泰国美食体验。。
